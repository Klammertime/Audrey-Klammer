Scrambler: jQuery Plugin Game



I created this game using JavaScript & jQuery. Unsplash images load and then are shuffled into either an Easy, Medium & Hard difficulty. These will only shuffle the blocks in a solveable way. If you need to see what the image looks like you can click on the Help button. When you've solved the game, the grid lines disappear. As a jQuery plugin, you can add this to any place on your site using a div and any image size you'd like.
